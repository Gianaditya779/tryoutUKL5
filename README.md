Program ini saya buat untuk menghitung rata rata nilai siswa di dalam suatu kelas.
Cara kerja Program ini adalah : 
1.User menginput jumlah siswa yang ada di kelas;
2.setelah jumlah siswa di dapat, user akan memasukkan nilai siswa satu persatu sejumlah dengan jumlah siswa yang di input;
3.Program akan menghitung nilai rata-rata dari siswa tersebut;
4.Selanjutnya program mengeluarkan output berupa hasil dari rata-rata nilai siswa di kelass.
